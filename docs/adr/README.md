# Registros de decisiones de arquitectura

Los ADR (Architecture Decision Records) documentan decisiones que afectan la
estructura, los limites o la forma de operar del sistema. Cada archivo conserva
el contexto, la decision y sus consecuencias.

Estados utilizados:

- `Aceptada`: decision vigente.
- `Reemplazada`: decision historica sustituida por otra ADR.
- `Descartada`: alternativa evaluada y no adoptada.
